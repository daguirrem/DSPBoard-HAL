enum gpio_config_t

GPIO configuration mask.

User defined mask to GPIO custom easy configuration.

Note
12 bit mask where:
0b CCCC x SS PP O MM
  • C: Configurator mask bits
  • S: Speed mask bits
  • P: Pull mode mask bits
  • O: Output mode mask bits
  • M: GPIO mode mask bits
  • x: none
Refer product datasheet for max frequency on speed levels pag. 117 - datasheet [Input/output AC characteristics].

Write logical one to single or multiple masked IOs on GPIO_BSRR[15:0] target register.

Parameters
[in]gpioTarget GPIO port.
[in]maskTarget output IO/s mask.
Returns
None
Note
1. To set multiple IOs you must specify an OR operator between IOs.
2. It is a forced inline function.

E.g.

  • For single output:
    ...
    // This set high the GPIOA[0] output.
    gpio_port_set(GPIOA, GPIO_IO0);
    ...
  • For multiple output:
    ...
    // This set high the GPIOA[0,2] output.
    gpio_port_set(GPIOA, GPIO_IO0 | GPIO_IO2);
    ...
