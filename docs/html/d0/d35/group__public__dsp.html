Inicializate static fir_t filter.

This macro create a fir_t variable with /n name of /o order on /s section of memory (optional). Also create an array for x of size /o and assign it to fir_t variable created. The order of the filter is saved on variable created.

Parameters
[in]nName of filter to create & inicializate.
[in]oOrder of filter to inicializate.
[in]s(Optional) Section where variables are created, it can be __CCM_VAR_I for CCM data or none for .data section.
Returns
None.
Note
1. After call this macro you can access to a new variable of name /n.
2. If you declarate in specific memory section, it must be called outside of main function (As global variable).
See also
fir_t

E.g.
This inicializate a static fir variable and then use them.

...
// B Coefficients array
float b[10] = {1,2,3,4,5,6,7,8,9};
...
int main(void){
...
// After that call of macro, you have declarated a fir_t variable
// with name 'filter'.
dsp_filter_fir_declarate(filter, 10);
// Now you can assign a B coefficient filter (For example High Pass)
filter.b.high = &b[0];
...
}

Executes a FIR on filter accordding the type.

dsp:filter:fir

Executes a convolution to compute a Y output of /filter and returns it, the execution depends of filter /type and it expect a last input of x.

y = x[0]*b./type[0] + x[1]*b./type[1] + ... x[order]*b./type[order].

Where: y is stored on /filter->y x is stored on /filter->x b is stored on /filter->b x[0] is the last input.

Parameters
[in]filterFilter to execute a FIR.
[in]typeFilter type to execute.
[in]floatLast input of x array of convolution.
Returns
Y computed output.

fir_t * dsp_filter_fir_init ( const uint16_t  order)

Inicializate a fir_t variable.

dsp:fir:init

Allocate memory to fir_t variable type on heap memory depending on the FIR /order.

The size of memory allocated is: fir_t type: 28 [bytes] + float x[order + 1]: (order + 1) * 4 [bytes]

The order filter is saved on /filter->order.

Parameters
[in]orderOrder to inicializate filter.
Returns
fir_t allocated.
Return values
NULLif allocation failed.
Otherwisethe address of fit_t allocated.
Note
1. The coefficients B are set to NULL pointer, you must to inicializate them by calling dsp_filter_fir_coeff_init() function.
2. Y output is set to 0.
3. This is used only for dynamic allocation.
