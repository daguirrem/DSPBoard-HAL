◆ rcc_ahb1_peripheral_clk_dis()

__FORCE_INLINE void rcc_ahb1_peripheral_clk_dis ( rcc_ahb1_t  mask)

Disable clock to AHB1 peripherials.

Allow disable clock to peripherials:

  • GPIO[A,B,C,D,E,F,G,H,I]
  • CRC
  • Backup SRAM interface
  • CCM Data RAM interface
  • DMA[1,2]
  • Ethernet [MAC, TX, RX, PTP]
  • USB [HS, HS_ULPI].
Parameters
[in]maskAHB1 Peripheral/s mask to be disabled. You can disable multiple peripherals at time passing each of them with an OR operator.
See also
rcc_ahb1_t
Note
It is a forced inline function.

E.g.
For one peripheral disable:

...
// This disable clock to GPIOA peripheral
rcc_ahb1_peripheral_clk_dis(RCC_AHB1_GPIOA);
...

rcc_ahb1_peripheral_clk_dis(RCC_AHB1_GPIOA | RCC_AHB1_GPIOB);
...
