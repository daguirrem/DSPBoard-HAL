gpio_alt_t  af,
gpio_io_t  mask 
)

Switch the alternate IO/s function on target GPIO.

gpio:configure:alternate_function

Configure the alternate function register GPIO_AFR of masked IO/s on target GPIO.

Parameters
gpioTarget GPIO port.
afAlternate function to switch.
maskGPIO Mask to configure.
Returns
None
Note
To alternate multiple IOs you must specify an OR operator between IOs.

E.g
For Single IO:

...
// This switch the alternate function of GPIOA[0] to AF2
gpio_config_af(GPIOA, GPIO_ALT_AF0, GPIO_IO0);
...
