enum usart_config_t

U(S)ART configuration mask.

User defined mask to U(S)ART custom easy configuration.

Note
12 bit mask where:
0b CCCC x SS PP O MM
  • C: Configurator mask bits
  • S: Speed mask bits
  • P: Pull mode mask bits
  • O: Output mode mask bits
  • M: GPIO mode mask bits
  • x: none
