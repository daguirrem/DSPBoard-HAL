void delay_init ( void  )

Init internal variables to delay.

delay:initialization

Init a DELAY_CYCLES_MS & DELAY_CYCLES_US variables to delay functions.

Returns
None.
Note
1. If a SystemCoreClock is updated, its required reinit the delay.
2. sys_clk_update executes a SystemCoreClock & Delay vars update.

E.g.

...
// This will update the SystemCoreClock variable & update delay vars
SystemCoreClockUpdate();
delay_init();
...
